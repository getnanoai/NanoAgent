# NanoAgent.CLI

`NanoAgent.CLI` installs the `nanoai` command as a .NET global or local tool.

## Install

```bash
dotnet tool install --global NanoAgent.CLI
```

## Update

```bash
dotnet tool update --global NanoAgent.CLI
```

## Run

```bash
nanoai
```
